# World-s-Metro-Systems-Database
A database of the world's metro systems (with some extra info)

Created using private third-party sources, updated with Wikipedia Tables.
Database includes:
  - Country
  - City
  - Name (System Name)
  - Length (km)
  - Number of lines
  - Stations
  - Opening (Date / DD/MM/YYYY)
  - Opening Ranking (#1, #2, etc.)
  - World Region (Geographically)

Included in the repo as a csv file, originally created manually in Numbers (Apple).
